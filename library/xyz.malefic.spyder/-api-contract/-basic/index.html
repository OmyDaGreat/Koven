Basic

abstract class Basic<Res>(path: String, method: Method = Method.GET) : ApiContract<Unit, Res, NoHeaders, NoHeaders>

A contract for a simple GET-style API endpoint (no request body, no custom headers).

Parameters

path

The path of the API endpoint. Automatically prefixed with "api/".

method

The HTTP method to use for the API endpoint. Default is Method.GET.

Type Parameters

Res

The Serializable type of the request body. If the response body is empty, this should be Unit.
