ApiContract

abstract class ApiContract<Req, Res, ReqH : HeaderProvider, ResH : HeaderProvider, PathP : PathProvider, QueryP : QueryProvider>(val path: String, val httpMethod: HttpMethod = HttpMethod.POST, val requiredRequestHeaders: List<HeaderField<*>> = emptyList(), val requiredResponseHeaders: List<HeaderField<*>> = emptyList(), val requestHeaderDecoder: HeaderField<ReqH> = NoHeaders as HeaderField<ReqH>, val responseHeaderDecoder: HeaderField<ResH> = NoHeaders as HeaderField<ResH>, val pathDecoder: PathField<PathP> = NoParams as PathField<PathP>, val queryDecoder: QueryField<QueryP> = NoParams as QueryField<QueryP>, val requestFormat: BodyFormat<Req>, val responseFormat: BodyFormat<Res>, val isProtected: Boolean = false)

A contract for an API endpoint shared between the client and server.

Parameters

path

The path of the API endpoint. Automatically prefixed with "api/".

httpMethod

The HTTP method to use for the API endpoint. Defaults to HttpMethod.POST.

requiredRequestHeaders

The set of required request header fields for the API endpoint.

requiredResponseHeaders

The set of required response header fields for the API endpoint.

requestHeaderDecoder

The HeaderField to decode the request headers.

responseHeaderDecoder

The HeaderField to decode the response headers.

pathDecoder

The PathField to decode the path parameters.

queryDecoder

The QueryField to decode the query parameters.

requestFormat

The BodyFormat to use for the request body. Defaults to UnitFormat or JsonFormat.

responseFormat

The BodyFormat to use for the response body. Defaults to UnitFormat or JsonFormat.

isProtected

Whether the API endpoint requires authentication. Defaults to false.

Type Parameters

Req

The type of the request body. If the request body is empty, this should be Unit.

Res

The type of the response body. If the response body is empty, this should be Unit.

ReqH

The HeaderProvider type of the request headers. Use NoHeaders for no headers.

ResH

The HeaderProvider type of the response headers. Use NoHeaders for no headers.

PathP

The PathProvider type of the path parameters. Use NoParams for no path parameters.

QueryP

The QueryProvider type of the query parameters. Use NoParams for no query parameters.
